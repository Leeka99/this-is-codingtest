# 모험가 길드

### 난이도 : 하
### 제한 시간 : 30분
### 나의 풀이 시간 : 시간 초과

---
#### key point
- 가장 많은 그룹의 수를 구하는 것이 포인트
- 공포도가 낮은 모험가부터 길드를 만들어야 최대한 많은 그룹을 만들 수 있다.
- 문제의 예시대로 (1,2,3) , (2,2) 이렇게 2 그룹으로 만드는 방법만 생각했다.
    - 다른 아이디어를 떠올리지 못했다.
- 정답 코드는 (1) , (2,2) 그룹으로 총 2개의 그룹을 만드는 것으로 구현되었다. 
    - 모든 모험가가 그룹에 참여할 필요는 없다.
