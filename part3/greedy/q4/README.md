# 만들 수 없는 금액

### 난이도 : 하
### 제한 시간 : 30분
### 나의 풀이 시간 : 시간 초과

---
#### key point
- 만들 수 없는 양의 정수가 나올 때까지 N 개의 동전을 누적시킨다.
```
    int target = 1;
    for (int i = 0; i < n; i++) {
        if (target < arr[i])
            break;
        target += arr[i];
    }
```
- for문을 이용해서 모든 조합을 다 계산하는 방법만 찾다가 시간안에 해결하지 못했다.
