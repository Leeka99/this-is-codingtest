#### key point
- 시간 복잡도 계산하면서 풀기
    - 현재 코드 : O(n^2) -> 약 100만
    - i 루프 : O(n) , j 루프 : : O(n / i), k 루프 : O(i)
    - 시간 복잡도 : O(n) * O(n/i) * O(i) => O(n^2)
- s.substring()
- String 을 + 하는 것은 느리다. 만약 데이터가 더 많다면 StringBuilder를 사용하자.
